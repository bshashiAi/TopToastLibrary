A professional, customizable top notification toast library for Android with beautiful animations and easy integration.

![Demo](screenshots/demo.gif)

## Features ✨

- 🎨 **5 Built-in Types**: Success, Error, Warning, Info, Normal
- 🎬 **Smooth Animations**: Slide, Fade, Bounce effects
- 🎯 **Fully Customizable**: Colors, icons, fonts, positions
- 📱 **AndroidX Support**: Modern Android development
- ⚡ **Lightweight**: Minimal footprint
- 🔧 **Builder Pattern**: Fluent API design
- 📦 **Easy Integration**: JitPack ready

## Installation 📥

### Step 1: Add JitPack Repository

**settings.gradle:**
```gradle
dependencyResolutionManagement {
    repositories {
        google()
        mavenCentral()
        maven { url 'https://jitpack.io' }
    }
}
